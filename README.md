## Linked List Application 
#### Integrated with:
- Binary Search
- Linear Search
- Bubble Sort
- Quick Sort

## Compile with g++
### Execute run.bat
1 - Change directory into inner project folder
```ps1
cd HotelReviewAnalysis
```
2 - Try one of these:
```ps1
run
run.bat
./run
./run.bat
```

### Compilation Command
```ps1
# cmd
g++ -Iinclude main.cpp lib/*.cpp -o HotelReviewAnalysis && HotelReviewAnalysis
# powershell
g++ -Iinclude main.cpp lib/*.cpp -o HotelReviewAnalysis && ./HotelReviewAnalysis
```
